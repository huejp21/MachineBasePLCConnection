
  모션 제어시 체크할 사항 

 1. 아진 보드 사용시 모터 파라미터는 아진 기본적용 파일 형식을 사용한다 mot 파일 사용 
 2. 아진 보드 사용시 가감속은 AxmMotSetAccelUnit 기본적으로 
    0 unit/sec2 사용 Sec = (최고속도 / 가속도) * 구동속도 ( 400 / 60 ) *100  // 값이 커지면 가감속이 더 빨라짐 
	1 sec 는 상황에 따라 사용  1000 입력시 1초동안 가감속을 하는것임 
 3. 아진 보드 사용시(PCI-N804/404) MPG를 사용하면 연속축을 사용못한다 예) 8축보드시 0번을 MPG 사용시 1번은 비워야 한다 
 4. 아진 보드 사용시 1D 보정 관련해서 gap값은 기본적으로 리턴이 안되는데 아진 담당자 한테 이야기 해서 라이브러리 업데이트를 하면 리턴을 할수 있다 (cmd 와 enc 를 같게 하려면 업데이트 해야함)
    (PCI-N804/404)는 포인트 제한이 없다. 
	메카트로 링크는 500점으로 제한이 있다. 
 5. 장비 PC에 프로그램을 적용했을때 폼 사이즈가 바뀌는경우 폼의 "AutoScaleMode" 속성을 "None" 으로 변경한뒤 장비 PC에 적용한다.
